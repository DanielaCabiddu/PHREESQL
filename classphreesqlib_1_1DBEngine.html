enum phreesqlib::DBEngine::EPSG_CONVERT_TYPE

The EPSG_CONVERT_TYPE enum.

The EPSG_CONVERT_TYPE enum enables the possibility to specify how to save the output of an EPSG conversion.

Enumerator
TABLE 

As a CSV file.

DB 

As a new table of the input PHREESQL database.

UNSET 

As a new database which is a copy of the input one with converted EPSG.
