  Main step:
1) git package
2) compile PHREESQLib
3) download tar.bz2 data and script 


- EPA_data: original files (from EPA Ireland) and pre-processed files are located here
- shell_scripts: folder with bash shell scripts
- sql_scripts: folder with SQL scripts
- run_DB: scratch folder where create PHREEQC input (IN folder), compute output (OUT folder), and creates metadata (META folder)
- DB: folder where SQL database are stored


How to run scripts and queries
PHREESQLIB and C&G_FOLDER should be in the same folder.
